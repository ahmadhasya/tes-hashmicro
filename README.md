# Test Hashmicro

This is Frontend Developer Test Assesment. 

Additional library for this projects are:
- Vuetify for material UI components
- Chart JS for chart components
- Pinia for state management

There are folders inside `/app`:
- `/components` contains chart component, product & worker edit dialog/modal, table component, and global alert like snackbar and loading.
- `/layouts` contains layouts for App
- `/pages` contains route app that available and can be accessed by browser
- `/plugins` contains plugin for App like vuetify
- `/stores` contains state managements using pinia for global loading and global snackbar
- `/utils` contains global method like `mixins` that can be called by components
